# Introduction to Python

A project serving as a gentle guide to Python programming exercises

## Tasks

### Task 0: Basic Arithmetic Exercise

- Objective: To practice basic arithmetic operations in Python by performing predefined calculations.

### Task 1: Simple Interest Calculator

- Objective: Calculate simple interest on a given principal amount, rate of interest and time with arithmetic operations and variable assignment.

### Task 2: Calculate the Area of a Rectangle

- Objective: Calculate the area of a rectangle using arithmetic operations and variables

### Task 3: Convert Hours to Seconds

- Objective: Convert hours to seconds using arithmetic operations and variable assignments

### Task 4: User Input Age Calculator

- Objective: Receive input from user and calculate user's age using arithmetic operations
